info Como funciona o cálculo:
Para cada opção, o sistema faz:
1. Base: Maior valor entre (Valor NF × % Frete da Cidade) e Valor Mínimo da Cidade.
2. Adicionais: Soma a Taxa Fixa + Pedágio + (Valor NF × % GRIS).
3. Excedente: Soma o (Peso Excedente × Valor Kg Excedente) se houver.
4. Imposto: O total acumulativo é dividido por (1 - % ICMS/100) para aplicar o cálculo "por dentro".
